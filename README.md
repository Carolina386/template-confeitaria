# 🍰 Landing Page - Confeitaria Concept

> Status do Projeto: 🚀 Concluído & Pronto para Produção

Landing Page responsiva desenvolvida para o nicho de confeitaria e docerias locais, projetada com foco em alta conversão visual, experiência do usuário (UX) e velocidade de carregamento.

## 🧠 Estratégia de Neurodesign & Negócios
Este projeto não foi construído focado apenas na estética, mas sim em **Psicologia Comportamental aplicada ao Código**:
* **Psicologia das cores (Paleta de Cores):** Uso estratégico de cores quentes (laranja/amber) para estimular o apetite e gerar senso de urgência na tomada de decisão.
* **UX de Conversão Rápida:** Arquitetura baseada em *Cards Tradicionais de Cardápio* e botões de chamada para ação (CTA) magnéticos direcionados para o WhatsApp/Delivery.
* **Acessibilidade & Escaneabilidade:** Textos de fácil leitura e hierarquia visual bem definida para garantir que o cliente encontre o produto e feche o pedido em menos de 3 cliques.

---

## 🛠️ Tecnologias Utilizadas

* **HTML5 Semantic:** Estrutura limpa e focada em SEO (otimização para motores de busca).
* **Tailwind CSS (via CDN):** Utilizado para estilização ágil, componentes modernos e responsividade nativa (Mobile-First).
* **Google Fonts:** Integração de tipografia específica para o nicho, melhorando a harmonia visual.
